# Support / 获取帮助

- 安装、配置和运行问题：在
  [Q&A Discussions](https://github.com/Baba88611/detroit-ai-player/discussions/categories/q-a)
  提问。
- 分享完整实验结果：前往
  [Show and tell](https://github.com/Baba88611/detroit-ai-player/discussions/categories/show-and-tell)。
- 可复现的代码缺陷：提交
  [Bug report](https://github.com/Baba88611/detroit-ai-player/issues/new?template=bug_report.yml)。
- 安全漏洞或敏感信息风险：按照 [SECURITY.md](SECURITY.md) 私密报告。

提问时请提供操作系统、Python 版本、使用的后端、已脱敏的命令和完整错误信息。
不要公开 API key、`.env` 内容或认证 token。
