## English

This is a **non-commercial research and commentary experiment** that studies
how different large language models (LLMs), acting as autonomous "players",
navigate the branching narrative of a story-driven game, comparing their
decision tendencies in moral dilemmas.

- *Detroit: Become Human* — its story, characters, scenes, dialogue, and all
  in-game content — is the trademark and copyrighted property of **Quantic
  Dream / Sony Interactive Entertainment**. This project is **not affiliated
  with, authorized, or endorsed by** them.
- This repository contains **no game assets**: no art, audio, video,
  executables, and **no original game script, dialogue transcripts, or
  verbatim excerpts of any kind**.
- The scene text in all 32 chapter decision-tree JSONs is an **original
  paraphrase written in the project author's own words** — it describes plot
  progression and decision situations rather than copying game text. The
  Chinese version is an independently written narrative summary; the English
  version underwent a **line-by-line dialogue audit** before release, with
  all original game dialogue rewritten as indirect narration or original
  phrasing. Plot outlines and branching structures are unprotectable ideas;
  their textual expression here is original to this project.
- This repository ships **no experiment results or analysis reports** — it is
  a framework for users to run their own AI models and observe their own
  results.
